# PYTHON PORT SCANNER
A simple tcp port scanner made by BashTheBalrog written in Python as a learning project

## Features
- Scan a single TCP port
- Scan a custom range of ports
- Display all/in-range open ports
- Resolves hostnames (e.g. google.com) to IP Addresses

## What stuff I used
- python 3
- socket module
- Git
- Github

## How to Run (the program ofc)

1. Clone the repo:
```bash
   git clone https://github.com/BashtheBalrog/port-scanner.git```

2. Open the project folder.

3. Run:
```bash
   python scanner.py```
